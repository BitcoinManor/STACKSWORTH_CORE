# 🟠 STACKSWORTH CORE

**Bitcoin’s Pulse — Simplified.**

The STACKSWORTH CORE is a compact Bitcoin display designed to bring essential Bitcoin data to life in a clean, accessible format.  ***STILL BUILDING THIS, Day 2

Built on an affordable 2.8" display platform, CORE serves as the perfect entry point into the STACKSWORTH ecosystem — with upgrade paths including custom CNC wood enclosures.

---

## 🔥 Features

- 📊 Real-time Bitcoin data display  
- ⚡ Fast and lightweight (optimized fetch intervals)  
- 📡 WiFi connectivity with captive portal setup  
- 🧠 Smart data caching for stability  
- 🎨 Clean UI optimized for small displays  
- 🪵 Optional premium wood enclosure (Walnut / Maple)

---

## 🧱 Hardware

- ESP32-based development board  
- 2.8" TFT LCD display  
- USB-powered  
- Optional custom CNC enclosure plates
- Drivers  #define ST7789_DRIVER or #define ILI9341_DRIVER based on your LCD display

---

## 📦 Versions

- **Acrylic Edition** – Includes stock top and bottom plates  
- **Walnut Edition** – CNC-cut premium hardwood  
- **Maple Edition** – CNC-cut light hardwood  

---

## ⚙️ Setup

1. Flash firmware using Arduino IDE  
2. Power on device  
3. Connect to WiFi via captive portal  
4. Enter your preferences (location, settings)  
5. Device begins displaying live Bitcoin data  

---

## 🌐 Data Sources

The CORE uses APIs from the STACKSWORTH backend:

- Bitcoin Price  
- Block Height  
- Fee Estimates  
- Miner Data *(optional depending on config)*
